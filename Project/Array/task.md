In Java, arrays let you store multiple values of the same type in a single variable. This is very useful when you want to process a list of data efficiently, like scores, temperatures, or items in a shop.

Loops help you process all elements of an array automatically instead of writing repetitive code. You can use a traditional for loop or a for-each loop, which is simpler when you don’t need the index.

Example: Track daily temperatures for a week and calculate the average.
